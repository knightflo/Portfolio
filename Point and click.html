Point and click game

This project is about moving the person to the white door in the castle. The person moves to the place where you click. There are obstacles hidden in this game. If you click on an obstacle, you will return to the start. This game was a learning project which I did on my own.

The game was my first experience with Javascript. I was very enthusiastic to discover that you can also create a game in a website. I also learned how to move an object on the screen.
