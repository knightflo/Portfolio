Troll game for mobile

This project is filled with traps. You have to move the red square to the end with the buttons on the screen. Learn to remember the locations of the traps.

I made this game on my own. It was a challenge finding out how to move the red square along the ground-floor to make jumps possible. There I needed some directions. I learned about the moving system for a mobile. I also learned how to switch screens so that it seems the screen follows the red block.
