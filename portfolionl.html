My first portfolio website

Om toegelaten te worden tot Sint-Lucas, heb ik mezelf voorgesteld met deze portfoliowebsite, waarin mijn projecten staan die ik voor september 2022 heb gemaakt. Er zijn games in verschillende programmeertalen op te vinden.

Een spel (Flappy Bob) heb ik gemaakt als onderdeel van een programmeercursus die ik in mijn vrije tijd in januari 2019 volgde. Andere spellen heb ik gemaakt met begeleiding van een programmeur in mijn woonplaats. Dit was de eerste keer dat ik een portfolio heb gemaakt en ik heb geleerd hoe ik mijn verschillende projecten op een website kan plaatsen en hoe je de layout kunt bepalen. Ook heb ik nagedacht over een eigen logo. In Flappy Bob heb ik geleerd te werken met een teller en in een ander spel genaamd Troll World heb ik geleerd hoe ik een 3D-wereld kan maken.
