Trolly adventures

This game is an obstacle run. The game has obstacles and hidden traps. As soon as you know where the traps are located, you can avoid them to finish the level. After 4 levels, you will beat the game.

This project is programmed with blocks instead of code. It is playful and easier than coding the game. But I learned that code has more possibilities. For instance, when you jump over the mushroom it should launch. With blocks, you can only check if an object passes the mushroom. With code, you can be more precisely: The launch only takes place when the player jumps over the mushroom. I made this game without help, I just studied YouTube tutorials.
