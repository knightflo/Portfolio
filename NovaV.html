NovaV

This game is about completing several quests. Most of the quests contains collecting items and bringing them back to the NPC's. The levels can be easy, but sometimes they can be difficult!

This game was created through a multidisciplinary collaboration involving three game developers and five game artists. My primary contribution was the development of the save system, which is capable of storing game data and retrieving it.
