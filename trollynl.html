Trolly adventures

Dit spel is een run met obstakels. Het spel heeft obstakels en verborgen valkuilen. Zodra je weet waar de valkuilen zich bevinden. Kun je ze vermijden om het level af te ronden. Na 4 levels heb je het spel afgerond.

Dit project is gemaakt met blokken in plaats van code. Het is speelser en makkelijker dan het spel coderen. Maar ik heb geleerd dat coderen meer mogelijkheden heeft. Bijvoorbeeld, als je over de paddenstoel springt, moet deze lanceren. Als je programmeert met blokken, kun je alleen controleren of een object de paddenstoel passeert. Met code kun je preciezer zijn: De lancering vindt alleen plaats als de speler over de paddenstoel springt. Ik heb dit spel zonder hulp gemaakt. Ik heb alleen YouTube tutorials bestudeert.
